# 💳 DevBank

Sistema de gestão financeira desenvolvido para auxiliar usuários no controle de receitas, despesas, transferências e integrações bancárias através de uma interface moderna, intuitiva e responsiva.

---

## 📖 Sobre o Projeto

O DevBank é uma aplicação Full Stack que permite ao usuário gerenciar sua vida financeira de forma simples e eficiente.

A plataforma oferece funcionalidades como:

- Controle de receitas e despesas
- Transferências financeiras
- Gerenciamento de cartões
- Integração com bancos
- Operações PIX
- Estatísticas financeiras
- Perfil e configurações personalizadas

## ⚙️ Como Executar

### Clonar o repositório

```bash
git clone https://github.com/Pabl0Hernandez/DevBank.git
```

### Entrar na pasta

```bash
cd DevBank
```

### Executar o Frontend

```bash
npm install
npm run dev
```

### Executar o Backend

```bash
dotnet restore
dotnet ef database update
dotnet run
```
